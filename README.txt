-Customer Retention Cohort Analysis-

I- Tools Used:
- SQL (PostgreSQL)
- Power BI
- DAX
- Python (optional data prep)

II- Project Overview:
- Extracted cohort data from ecommerce transaction table using PostgreSQL
- Calculated cohort start dates and weekly offsets
- Built Cohort Retention Matrix in Power BI
- Created DAX measures to calculate:
  - Total Users
  - Cohort Size
  - Week 1 Retention %
  - Long Term Retention %

III- Date Range:
- 2010 - 2011 ecommerce data

IV- Files Included:
- `Customer-Retention-Cohort-Analysis.pbix` (Power BI dashboard)
- `SQL_Cohort_Query.sql` (Cohort calculation SQL query)
- Sample screenshots
